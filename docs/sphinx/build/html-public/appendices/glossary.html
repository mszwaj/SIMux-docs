Glossary

Technical terms and acronyms used in SIMux documentation.

A

ARM Cortex-M0+

32-bit RISC processor core architecture used in SAMD21 microcontroller. Low power, suitable for embedded applications.

B

Baud Rate

Speed of serial communication measured in bits per second (bps). SIMux uses 115200 baud for USB serial.

Bootloader

Small program that allows firmware updates via USB without external programmer. SAMD21 has built-in UF2 bootloader activated by double-tapping RESET.

BOSSA

Basic Open Source SAM-BA Application - tool for flashing firmware to Atmel/Microchip ARM microcontrollers like SAMD21.

BOM (Bill of Materials)

List of components required to manufacture hardware.

C

CLI (Command Line Interface)

Text-based interface for controlling SIMux via serial terminal. Commands like SIM0=1, INFO, etc.

COM Port

Communication port identifier in Windows (e.g., COM3, COM5). Equivalent to /dev/ttyACM0 on Linux.

D

Daisy Chain

Method of connecting multiple devices in series. SIMux boards connect via Grove cables in I2C daisy chain topology.

E

EEPROM (Electrically Erasable Programmable Read-Only Memory)

Non-volatile memory that retains data when power is off. SIMux uses emulated EEPROM in flash for storing custom serial numbers and settings.

F

FPC (Flexible Printed Circuit)

Flexible cable with printed conductors. SIMux uses 6-pin 1.0mm pitch FPC to connect to modem carrier boards.

G

GPIO (General Purpose Input/Output)

Configurable digital pins on microcontroller. SIMux uses GPIO for I2C, LED control.

Grove Connector

Standardized 4-pin connector system by Seeed Studio. SIMux uses Grove for I2C daisy chaining between boards.

H

Hot-Swap

Ability to change SIM card while target device powered.

I

I2C (Inter-Integrated Circuit)

Two-wire serial communication protocol (SDA, SCL). Used for multi-board communication in SIMux (up to 8 boards).

I2C Address

7-bit identifier for I2C devices (0x20-0x27 for SIMux boards). Configured via A0, A1, A2 address pins.

ICC (Integrated Circuit Card)

Alternative term for SIM card. Refers to physical smart card containing subscriber identity.

ICCID (Integrated Circuit Card Identifier)

Unique 19-20 digit serial number printed on SIM card. Not readable by SIMux.

IMSI (International Mobile Subscriber Identity)

Unique 15-digit identifier stored on SIM card. Used for network authentication. Not readable by SIMux.

ISO/IEC 7816-3

International standard defining electrical interface and transmission protocols for smart cards including SIM cards.

J

JLCPCB

PCB manufacturer offering assembly services. SIMux PCBs can be manufactured and assembled by JLCPCB.

JSON (JavaScript Object Notation)

Data interchange format. SIMux supports JSON commands for programmatic control.

K

KiCad

Open-source PCB design software. SIMux hardware designed in KiCad 9.0.

L

LED (Light Emitting Diode)

Indicator lights on SIMux:

  • Power LED (green): Power indicator

  • User LED (orange/amber): Status indicator

  • SIM LEDs (yellow): Active SIM indicator (4 LEDs for slots 1-4)

M

MCU (Microcontroller Unit)

Integrated circuit containing processor, memory, peripherals. SIMux uses Seeeduino XIAO SAMD21 (ARM Cortex-M0+).

Modem

Device that modulates/demodulates radio signals for cellular communication. SIMux connects between modem and SIM cards.

Multiplexer

Device that selects one of multiple inputs to route to output.

MyST (Markedly Structured Text)

Markdown parser used by Sphinx. This documentation uses MyST for source files.

N

nanoSIM

Smallest physical SIM card format (12.3mm × 8.8mm × 0.67mm). SIMux uses nanoSIM slots exclusively.

NVM (Non-Volatile Memory)

Memory that persists without power. SAMD21 flash acts as NVM for EEPROM emulation.

O

OTG (On-The-Go)

USB feature allowing devices to act as host. Required for connecting SIMux to Android phones/tablets.

P

PCB (Printed Circuit Board)

Board with copper traces connecting components. SIMux is a 4-layer PCB.

PCBA (Printed Circuit Board Assembly)

PCB with components soldered. JLCPCB provides PCBA services for automated assembly.

PID (Product ID)

USB identifier for device type. SIMux inherits PID from SAMD21 bootloader (0x802F).

PlatformIO

Cross-platform build system for embedded development. Recommended for building SIMux firmware.

Pull-up Resistor

Resistor connecting signal line to positive voltage. I2C requires pull-ups on SDA/SCL lines (4.7kΩ typical).

R

RESET

MCU reset - clears CPU state and restarts firmware. Triggered by RESET command or power cycling.

RTD (Read the Docs)

Documentation hosting platform. SIMux uses Read the Docs theme for Sphinx.

S

SAMD21

Microchip (formerly Atmel) ARM Cortex-M0+ microcontroller. Features 256KB flash, 32KB RAM, 48MHz clock.

SDA (Serial Data)

I2C data line. Bidirectional signal for I2C communication.

SCL (Serial Clock)

I2C clock line. Clock signal generated by I2C master.

Serial Number

Unique identifier for USB device. SIMux supports custom serial numbers via SETSN command (stored in EEPROM).

SIM (Subscriber Identity Module)

Smart card storing subscriber identity for cellular networks. SIMux switches between 4 nanoSIM cards per board.

Sphinx

Python documentation generator. Used to build this SIMux documentation from markdown/RST source files.

U

UID (Unique Identifier)

128-bit unique serial number burned into SAMD21 during manufacturing. Readable via INFO command.

USB CDC (Communications Device Class)

USB device class for serial communication. SIMux appears as virtual COM port using CDC.

USB-C

SIMux uses USB-C for power and communication.

V

VID (Vendor ID)

USB vendor identifier. SIMux uses Seeed Studio VID (0x2886).

X

XIAO

Seeed Studio’s tiny development board series. SIMux uses Seeeduino XIAO SAMD21 module.
