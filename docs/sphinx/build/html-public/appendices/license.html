Privacy

Data Collection

SIMux Does NOT:

  • Transmit data over network (no WiFi/Ethernet)

  • Log sensitive information (SIM card data, IMSI, ICCID)

  • Store user data externally

  • Phone home or check for updates

Local Storage Only:

  • Custom USB serial number stored in EEPROM (user-configurable)

  • SIM state persists locally on device

  • No cloud services or remote access
